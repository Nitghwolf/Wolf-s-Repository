Мои работы на Python

  • Консольные программы

    • Игра в БлэкДжек

    • Игра в позволяет выбирать количество игроков от 2 до 4. Игра идёт по очереди, каждый раз программа спрашивет игрока берёт он карту или нет. Затем считывает набранные очки и в конце выдает результат и победителя.

    • Ссылка на код на GitHub

    • Подсчёт процентов вклада

    • Программа запрашивает процент вклада, сумму вклада и срок вклада. Высчитывает и выдает результат - общую сумму вклада, с накопившимся процентами на конец срока вклада.

    • Код программы:

    • Подсчёт слов

    • Программа запращивает путь к текстовому файлу. Затем открывает его, удаляет знаки пунктуации и переходы на новую строку. Приравнивает всё к нижнему регистру и разбивает на слова. Создаёт словарь из данных слов и подсчитывает их. Результатом выводит количество всех слов, уникальных слов и выводит все использованные слова в документе.

    • Ссылка на код на GitHub

  • Программы с графическим интерфейсом

    • Список

    • Программа выводит окно в котором можно вписывать новые языки или любые слова и добавлять их в список. Из списка можно удалить слово по нажатию кнопки "Удалить".

    • Ссылка на код на GitHub

    • Конвертор температуры

    • Программа выводит окно в котором можно ввести температуру в Фаренгейтах и по нажатию кнопки конвертирует её в температуру по Цельсию.

    • Ссылка на код на GitHub

    • Отгадай слово

    • Программа выводит окно в котором загаданно одно слово из списка. Надо вписать это слова, в случае неудачи выводиться окошко с ообщением "Вы не угадали", в случае удачи окошко с собщением "Вы угадали".

    • Ссылка на код на GitHub

  • Программы на QtPy и Designer

    • Тест

    • Программа выводит окно с тремя вопросами, чтобы открыть следующий вопрос нужно правильно ответить на текущий.

    • Ссылка на код на GitHub

    • Guess Word

    • Программа выводит окно в котором можно ввести произвольный набор букв и выбрать из 4 или 5 букв будут состоять слова. Затем программа из подключенного словаря методом перебора введеных букв находит все слова, которые можно составить из этих букв. И выводит результат: количество слов, сами слова, количество комбинаций перебора и время которое на это затратилось.

    • Ссылка на код на GitHub

    • Анализатор

    • Программа выводит окно в котором можно открыть папку с результатами тестирования и в окно выведеться статистика выполнения тестирования по студентам. С возможностью сортировать по нужной строке или столбцу.

    • Так же можно открыть и вывести статистику по самим вопросов.
